# Sentiment Classification from Movie Reviews

Ένα από τα σημερινά προβλήματα των δεδομένων μεγάλου όγκου και συγκεκριμένα
της εξόρυξης δεδομένων από κείμενα (Text Mining) είναι η εξαγωγή και ανάλυση του
συναισθήματος (Sentiments) από κείμενα και συγκεκριμένα από κείμενα σχολίων
χρηστών μιας πλατφόρμας όπως Youtube κλπ.

Στόχος της εργασία αυτής είναι να σχεδιάσουμε έναν ταξινομητή για δεδομένα
κειμένου τα οποία προέρχονται από σχόλια τα οποία γίνονται για ταινίες.

## Δεδομένα:

Τα δεδομένα τα οποία θα χρησιμοποιήσετε βρίσκονται στην Movie Review Data
(http://www.cs.cornell.edu/people/pabo/movie-review-data/). Θα πρέπει να κατεβάσετε
το αρχείο polarity dataset v2.0 το οποίο και περιέχει 1000 αρνητικά σχόλια και 1000
θετικά σχόλια.
